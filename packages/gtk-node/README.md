# gtk-node

This project is a work in progress

## System Dependencies
- gtk+-3.0
- gtkmm-3.0
- sigc++-2.0

## Setup
```
npm install
```

## Run the example
```
npm run example
```

## Docs
The docs are automatically generated from a typescript declaration file that NBIND builds for us!
```
npm run build:docs # outputs into ./docs/
```
